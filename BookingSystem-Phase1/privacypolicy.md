# Privacy Policy

## 1. Data Controller
The booking system is operated by the service provider, who acts as the data controller under GDPR.

## 2. Personal Data Collected
The system processes the following personal data:
- Email address
- Password (stored in hashed form)
- Age (for age verification)
- User role (administrator or reserver)
- Booking information

## 3. Purpose of Processing
Personal data is processed to:
- Register and authenticate users
- Enable resource booking
- Manage resources and reservations
- Maintain system security

## 4. Legal Basis
The legal basis for processing personal data is the performance of a contract under GDPR Article 6(1)(b).

## 5. Data Minimization
Only data necessary for operating the booking system is collected.  
No marketing or analytics data is processed.

## 6. Data Retention
Personal data is stored only while the user account is active or as required by law.

## 7. Data Disclosure
Booking information is visible publicly without revealing personal data.  
Personal data is not shared with third parties.

## 8. Data Security
Technical and organizational measures are used to protect personal data, including authentication and access control.

## 9. Rights of the Data Subject
Users have the right to:
- Access their personal data
- Request correction or deletion
- Restrict or object to processing
- Data portability where applicable

## 10. Contact
Requests related to personal data should be directed to the system administrator.
